# Olimpíadas

Este projeto foi desenvolvido durante um curso na [Alura](https://www.alura.com.br/) e consiste em uma aplicação web que apresenta dados sobre as Olimpíadas.

## Tecnologias Utilizadas

- **HTML**: Estruturação das páginas web.
- **CSS**: Estilização e layout das páginas.
- **JavaScript**: Funcionalidades e manipulação de dados.

## Estrutura do Projeto

- `cursoalura.html`: Página principal da aplicação.
- `style.css`: Arquivo de estilos para a página.
- `app.js`: Script principal que controla a lógica da aplicação.
- `dados.js`: Contém os dados utilizados na aplicação.

## Funcionalidades

- Exibição de informações sobre as Olimpíadas.
- Interatividade através de elementos dinâmicos implementados com JavaScript.
